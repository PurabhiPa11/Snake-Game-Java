# 🐍 Snake Game (Java)

A simple and interactive Snake Game built using Java Swing.  
The game features smooth movement, real-time controls, and a restart option.

---

## 🚀 Features
- Snake movement using arrow keys
- Food generation at random positions
- Collision detection (walls + self)
- Game Over screen
- Restart button to play again
- Clean and responsive GUI using Java Swing

---

## 🛠️ Technologies Used
- Java
- Swing (GUI)
- AWT (Graphics & Events)
