# Snake Game (Java)

This project is a classic Snake Game built using Java Swing.  
It includes score tracking, high score system, increasing speed, and restart functionality.

---

##  Overview

The game allows the player to control a snake using arrow keys.  
The snake grows as it eats food, and the game ends if the snake hits the wall or itself.

---

##  Features

- Classic Snake Game mechanics  
- Score system (increments by 10)  
- High Score tracking during runtime  
- Increasing speed as score increases  
- Restart button after game over  
- Smooth keyboard controls (arrow keys)  
- Clean UI using Java Swing  

---

##  How to Play

- Use arrow keys to control the snake  
- Eat the red food to grow and increase score  
- Avoid hitting walls or your own body  
- Click **Restart** after game over to play again  

---

##  Technologies Used

- Java  
- Java Swing (GUI)  
- AWT (Graphics & Events)

---
