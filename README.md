# Snake Game (Java GUI)

This project is a classic Snake Game built using Java Swing.  
It provides a simple graphical interface where the player controls the snake using keyboard arrow keys.

## 🧩 Overview
The snake moves continuously on a grid. The player must guide it to eat food while avoiding collisions with walls and itself.  
Each time the snake eats food, it grows longer, making the game more challenging.

## ✨ Features
- Smooth snake movement using arrow keys
- Random food generation
- Snake grows after eating food
- Collision detection (wall + self)
- Game Over screen
- Simple and clean GUI

## 🛠️ Technologies Used
- Java
- Swing (GUI)
- AWT Event Handling
